# 📈 주식 스크리닝 & 백테스팅 시스템 사용자 매뉴얼

**버전**: 1.0.0
**업데이트**: 2025-09-24
**대상**: 한국 주식시장 투자자

## 🎯 시스템 개요

이 시스템은 32개의 기술적 지표를 활용하여 주식을 스크리닝하고, 선별된 종목들의 과거 성과를 백테스팅할 수 있는 종합 분석 도구입니다.

### 주요 기능
- **다중 조건 스크리닝**: 여러 기술지표 조건을 동시에 적용
- **32개 기술지표**: RSI, MACD, 볼린저밴드 등 전문가급 지표 활용
- **포트폴리오 백테스팅**: 실제 투자 성과 시뮬레이션
- **위험도 분석**: 샤프비율, 최대손실폭 등 리스크 지표 제공

## 📊 1단계: 기술지표 이해하기

### 주요 지표 카테고리

#### 🔹 추세 지표 (Trend Indicators)
| 지표 | 설명 | 골든크로스 조건 |
|------|------|----------------|
| **SMA** | 단순이동평균 | 단기 SMA가 장기 SMA를 상향 돌파 |
| **EMA** | 지수이동평균 | 단기 EMA가 장기 EMA를 상향 돌파 |
| **MACD** | 이동평균수렴확산 | MACD선이 시그널선을 상향 돌파 |

#### 🔹 모멘텀 지표 (Momentum Indicators)
| 지표 | 설명 | 추천 임계값 |
|------|------|-------------|
| **RSI** | 상대강도지수 | 30 이하(과매도), 70 이상(과매수) |
| **Stoch** | 스토캐스틱 | 20 이하(과매도), 80 이상(과매수) |
| **CCI** | 상품채널지수 | -100 이하(과매도), +100 이상(과매수) |

#### 🔹 변동성 지표 (Volatility Indicators)
| 지표 | 설명 | 활용 방법 |
|------|------|-----------|
| **BB** | 볼린저밴드 | 밴드 하단 터치 시 매수 신호 |
| **ATR** | 평균 진폭 | 높을수록 변동성 큰 종목 |

#### 🔹 거래량 지표 (Volume Indicators)
| 지표 | 설명 | 활용 방법 |
|------|------|-----------|
| **OBV** | 온밸런스볼륨 | 상승 시 매수세 강함 |
| **MFI** | 자금흐름지수 | 20 이하(과매도), 80 이상(과매수) |
| **VWAP** | 거래량가중평균가격 | 현재가와 비교하여 매수/매도 판단 |

## 🔍 2단계: 스크리닝 조건 설정하기

### 조건 유형 설명

#### 1️⃣ 골든크로스 (Golden Cross)
- **의미**: 단기선이 장기선을 상향 돌파
- **설정법**: 조건에서 "골든크로스" 선택
- **조회기간**: 최근 며칠 내 발생 (추천: 3-10일)

**예시 - RSI 골든크로스**
```
지표: RSI
조건: 골든크로스
조회기간: 5일
→ 최근 5일 내 RSI가 상승 전환한 종목 검색
```

#### 2️⃣ 데드크로스 (Death Cross)
- **의미**: 단기선이 장기선을 하향 돌파
- **활용**: 하락 신호, 공매도 전략

#### 3️⃣ 임계값 이상/이하
- **의미**: 지표값이 특정 수치를 넘거나 밑도는 경우

**RSI 과매도 구간 예시**
```
지표: RSI
조건: 임계값 이하
값 1: 30
→ RSI가 30 이하인 과매도 종목 검색
```

#### 4️⃣ 값 범위 (Between)
- **의미**: 지표값이 특정 범위 내에 있는 경우

**RSI 중립 구간 예시**
```
지표: RSI
조건: 값 범위
값 1: 40
값 2: 60
→ RSI가 40~60 사이인 중립 종목 검색
```

### 📋 추천 스크리닝 전략

#### 🟢 **모멘텀 돌파 전략**
```
조건 1: RSI 골든크로스 (조회기간: 5일)
조건 2: MACD 골든크로스 (조회기간: 3일)
조건 3: ADX 임계값 이상 (값: 25)
```
→ **강한 상승 모멘텀을 보이는 종목 선별**

#### 🔵 **과매도 반등 전략**
```
조건 1: RSI 임계값 이하 (값: 25)
조건 2: MFI 임계값 이하 (값: 20)
조건 3: CCI 임계값 이하 (값: -100)
```
→ **과도하게 하락한 종목의 반등 기회 포착**

#### 🟡 **안정적 추세 전략**
```
조건 1: RSI 값 범위 (값1: 40, 값2: 60)
조건 2: ADX 임계값 이상 (값: 20)
조건 3: MACD 골든크로스 (조회기간: 7일)
```
→ **과열되지 않은 상태에서 추세가 시작되는 종목**

## 💰 3단계: 백테스팅 설정하기

### 주요 설정값 가이드

#### 📅 **기간 설정**
- **시작일/종료일**: 분석하고 싶은 투자 기간
- **추천 기간**: 최소 1개월 ~ 최대 1년
- **주의사항**: 너무 짧으면 우연의 결과, 너무 길면 시장 환경 변화 무시

#### 💵 **초기 자본**
- **기본값**: 1,000,000원 (100만원)
- **설정 팁**: 실제 투자 가능 금액으로 설정
- **영향**: 절대 수익 계산에만 영향, 수익률은 동일

#### 📊 **포지션 크기**
- **의미**: 전체 자본 중 각 종목에 투자할 비율
- **기본값**: 0.1 (10%)
- **설정 가이드**:
  - `0.05 (5%)`: 매우 보수적, 20개 종목 분산
  - `0.1 (10%)`: 표준적, 10개 종목 분산
  - `0.2 (20%)`: 공격적, 5개 종목 집중
  - `0.33 (33%)`: 고위험, 3개 종목 집중

#### 💳 **거래 비용**
- **의미**: 매수/매도 시 발생하는 수수료 + 세금
- **기본값**: 0.003 (0.3%)
- **한국 주식 실제 비용**:
  - **매수**: 수수료 0.015% + 기타 0.01% = 약 0.025%
  - **매도**: 수수료 0.015% + 증권거래세 0.25% + 기타 = 약 0.3%
  - **총 비용**: 약 0.3% 추천

## 📈 4단계: 결과 해석하기

### 포트폴리오 성과 지표

#### 🎯 **수익률 지표**
| 지표 | 설명 | 좋은 수준 |
|------|------|-----------|
| **총 수익률** | 전체 투자기간 수익률 | 연 10% 이상 |
| **연환산 수익률** | 1년 기준 환산 수익률 | 시장수익률 초과 |

#### ⚖️ **위험도 지표**
| 지표 | 설명 | 좋은 수준 |
|------|------|-----------|
| **변동성** | 수익률의 표준편차 | 낮을수록 안정적 |
| **샤프 비율** | 위험 대비 수익 비율 | 1.0 이상 우수 |
| **최대 손실폭** | 최고점 대비 최대 하락률 | 20% 이하 양호 |

#### 📊 **해석 예시**

**좋은 결과 예시**
```
총 수익률: +15.2%
샤프 비율: 1.35
최대 손실폭: -12.4%
성공률: 70%
```
→ **위험 대비 좋은 수익, 안정적인 전략**

**주의 필요한 결과**
```
총 수익률: +25.8%
샤프 비율: 0.6
최대 손실폭: -35.2%
성공률: 40%
```
→ **높은 수익이지만 위험도 과도, 전략 수정 필요**

## 🛠️ 실전 활용 가이드

### Step 1: 시장 상황 파악
1. **상승장**: 모멘텀 돌파 전략 활용
2. **하락장**: 과매도 반등 전략 활용
3. **박스권**: 안정적 추세 전략 활용

### Step 2: 다중 조건 조합
```
기본 전략 + 필터링 조건
예: RSI 골든크로스 + ADX > 25 + 거래량 급증
```

### Step 3: 백테스팅으로 검증
1. **과거 6개월 데이터**로 전략 테스트
2. **서로 다른 기간**으로 재테스트
3. **일관된 결과** 확인 후 실전 적용

### Step 4: 포트폴리오 구성
1. **스크리닝으로 종목 선별** (10-20개)
2. **백테스팅으로 성과 검증**
3. **상위 5-10개 종목**으로 포트폴리오 구성
4. **정기적 리밸런싱** (월 1회 권장)

## ⚠️ 주의사항 및 한계

### 🚨 **투자 위험 고지**
- 과거 성과가 미래 수익을 보장하지 않음
- 백테스팅은 실제 거래와 차이가 있을 수 있음
- 반드시 손절매 기준을 설정하여 운용
- 투자 금액은 여유자금 내에서 설정

### 📋 **시스템 한계**
- **생존편향**: 상장폐지 종목 미반영
- **거래량 제한**: 실제 거래 가능 여부 미고려
- **슬리피지 미반영**: 실제 체결가격과 차이 가능
- **배당금 미포함**: 배당 수익 별도 고려 필요

### 🔧 **권장 사항**
1. **소액으로 시작**: 전략 검증 후 투자 규모 확대
2. **정기 모니터링**: 월 1회 이상 성과 점검
3. **전략 다변화**: 단일 전략에 의존하지 말 것
4. **손실 제한**: 개별 종목당 손실 한도 설정 (-10% 권장)

## 📞 문제해결 FAQ

### Q1: 스크리닝 결과가 너무 많이 나와요
**A**: 조건을 더 엄격하게 설정하세요
- 조회기간을 줄이기 (10일 → 3일)
- 임계값을 더 극단적으로 설정
- 추가 필터 조건 적용

### Q2: 스크리닝 결과가 하나도 없어요
**A**: 조건을 완화하세요
- 조회기간을 늘리기 (3일 → 10일)
- 임계값을 덜 극단적으로 설정
- 조건 개수 줄이기

### Q3: 백테스팅 수익률이 음수로 나와요
**A**: 다음을 확인하세요
- 분석 기간이 시장 하락기인지 확인
- 거래 비용이 과도하지 않은지 점검 (0.3% 권장)
- 다른 기간으로 재테스트

### Q4: 어떤 지표 조합이 좋은가요?
**A**: 서로 다른 카테고리 지표 조합 권장
- 추세 + 모멘텀: MACD + RSI
- 모멘텀 + 거래량: RSI + MFI
- 추세 + 변동성: SMA + ATR
