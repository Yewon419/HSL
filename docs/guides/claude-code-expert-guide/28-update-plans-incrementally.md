# Project Management #28: 계획 점진적 업데이트

## 원칙
계획을 업데이트하고, 채팅을 정리한 후, 계획을 참조하여 계속 진행하세요.

## 워크플로우
```
1. 초기 계획 작성
   "10단계 계획을 docs/plan.md에 저장해줘"

2. Step 1-3 완료

3. 계획 업데이트
   "@docs/plan.md에서 1-3을 완료로 체크하고,
   Step 4를 세부 계획으로 확장해줘"

4. 컨텍스트 정리
   /clear

5. 계속 진행
   "@docs/plan.md의 Step 4를 진행해줘"
```

## 장점
- 진행 상황 추적
- 컨텍스트 최적화
- 긴 프로젝트 관리 용이
