# Project Management #23: 세션당 한 가지 기능

## 원칙
한 세션을 하나의 프로젝트나 기능에 집중시켜 모든 컨텍스트가 관련성을 유지하도록 하세요.

## ✅ 좋은 스코핑
```
Session 1: 사용자 인증 구현
Session 2: 결제 통합
Session 3: 이메일 알림
```

## ❌ 나쁜 스코핑
```
Session 1:
- 사용자 인증
- 결제 통합
- 이메일 알림
- 프론트엔드 수정
- 버그 수정
→ 컨텍스트 혼란
```

## 전환 시
```
기능 완료 → /clear → 새 기능 시작
```
