- 매수: 2025-03-28, 121,000원 (RSI: 28.06)
- 매도: 2025-06-12, 95,500원 (RSI: 71.88)
- 손실: **-21.07%**

#### 손실 원인
1. ✗ **전체 하락 추세**: 기간 중 주가 -9.46% 하락
2. ✗ **Dead Cat Bounce**: RSI < 30 매수 후 일시 반등, 이후 급락
3. ✗ **MA20 < MA50**: 하락 추세 (104,720원 < 106,286원)
4. ✗ **변동성 높음**: 일일 변동성 3.29%

---

### 공통 손실 원인

#### 주요 문제점
1. **하락 추세 무시**
   - RSI < 30이 "저가 매수 기회"가 아니라 "추가 하락 신호"
   - MA20 < MA50 하락 추세에서도 무조건 매수

2. **손절 부재**
   - RSI > 70까지 기다리다가 큰 손실 확정
   - 매수가 대비 -20% 이상 손실에도 보유

3. **Falling Knife 현상**
   - 극단적 RSI(7.86)도 바닥이 아님
   - 추가 하락 가능성 간과

4. **추세 확인 없음**
   - 단순 RSI 값만으로 판단
   - 이동평균선, MACD 등 보조지표 미활용

## 3️⃣ 개선된 RSI 전략

### 개선 사항

#### 1. 추세 필터 추가
```
매수 조건: RSI < 30 AND MA20 > MA50 (상승 추세)
```

#### 2. 손절선 설정
```
매수가 대비 -5% 하락 시 무조건 손절
```

#### 3. RSI 임계값 강화
```
매수: RSI < 25 (더 극단적)
매도: RSI > 75 (더 극단적)
```

#### 4. 거래량 확인
```
RSI < 30 + 거래량 급증(1.5배 이상) 시에만 매수
```

#### 5. MACD 보조 지표
```
MACD 골든크로스 시 가점
```

1. ✅ RSI < 30 (과매도)
2. ✅ MA20 > MA50 (상승 추세)
3. ✅ 거래량 증가 (평균 대비 1.2배 이상)

#### 가점 조건

#### 분할 매수

### 매도 전략

#### 익절 조건
1. RSI > 75 (과매수)
2. 목표 수익률 도달 (15-20%)
3. MA20 < MA50 (추세 전환)

#### 손절 조건
- **필수**: 매수가 대비 -5% 도달 시 무조건 손절
